## Configuration (Env variables)
### Create a .env file in the root directory with the following env variables
```
PORT=4000
CORS_ORIGIN (For Production Deploy)
```

## To start the Backend
```
cd Cleanomatics_Solutions_backend
npm i
npm run dev
```
